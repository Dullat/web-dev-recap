Complete CSS Animations Guide

1. Transitions - The Foundation

Transitions create smooth changes between states. They're perfect for hover effects and simple animations.

Try hovering over this box:

.transition-basic { background: #ff6b6b; transition: all 0.3s ease; } .transition-basic:hover { background: #4ecdc4; transform: scale(1.1); border-radius: 50%; }

Transition Properties:

  • transition-property: What to animate (all, background, transform, etc.)
  • transition-duration: How long (0.3s, 500ms, 2s)
  • transition-timing-function: Speed curve (ease, linear, ease-in-out)
  • transition-delay: Wait time before starting

7. Complex Animation - Orbiting Elements

@keyframes orbit { from { transform: rotate(0deg) translateX(90px) rotate(0deg); } to { transform: rotate(360deg) translateX(90px) rotate(-360deg); } } .orbit { animation: orbit 4s linear infinite; } .orbit:nth-child(1) { animation-delay: 0s; } .orbit:nth-child(2) { animation-delay: -1s; } .orbit:nth-child(3) { animation-delay: -2s; } .orbit:nth-child(4) { animation-delay: -3s; }

8. Performance & Best Practices

🚀 Performance Tips:

  • Use transform and opacity - They're GPU accelerated
  • Avoid animating: width, height, top, left (causes layout)
  • Use will-change: transform; for complex animations
  • Remove animations when not visible (use animation-play-state: paused)
/* Good - GPU accelerated */ .fast { transform: translateX(100px); opacity: 0.5; transition: transform 0.3s, opacity 0.3s; } /* Avoid - causes layout/paint */ .slow { left: 100px; width: 200px; background: red; } /* Optimize complex animations */ .complex { will-change: transform; }

Animation Principles:

  • Easing: Natural motion feels better than linear
  • Duration: 200-500ms for UI, longer for attention-grabbing
  • Purposeful: Animations should guide users or provide feedback
  • Accessible: Respect prefers-reduced-motion for sensitive users
/* Respect accessibility preferences */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } }

9. Common Animation Patterns

/* Fade In */ @keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } /* Slide Up */ @keyframes slideUp { from { transform: translateY(30px); opacity: 0; } to { transform: translateY(0); opacity: 1; } } /* Scale In */ @keyframes scaleIn { from { transform: scale(0.8); opacity: 0; } to { transform: scale(1); opacity: 1; } } /* Bounce In */ @keyframes bounceIn { 0% { transform: scale(0.3); opacity: 0; } 50% { transform: scale(1.05); } 70% { transform: scale(0.9); } 100% { transform: scale(1); opacity: 1; } } /* Shake */ @keyframes shake { 0%, 100% { transform: translateX(0); } 10%, 30%, 50%, 70%, 90% { transform: translateX(-10px); } 20%, 40%, 60%, 80% { transform: translateX(10px); } }
