1. Object-Fit (How images fit in containers)

cover (default)

object-fit: cover;

contain

object-fit: contain;

fill

object-fit: fill;

2. Filters (Visual effects)

Blur

filter: blur(3px);

Brightness

filter: brightness(1.5);

Grayscale

filter: grayscale(100%);

Hue Rotate

filter: hue-rotate(90deg);

Drop Shadow

filter: drop-shadow(10px 10px 20px rgba(0,0,0,0.5));

Combined Filters

filter: sepia(50%) hue-rotate(180deg) saturate(1.5);

3. Clip-Path (Custom shapes)

Circle

clip-path: circle(50%);

Triangle

clip-path: polygon(50% 0%, 0% 100%, 100% 100%);

Diamond

clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);

Hexagon

clip-path: polygon(30% 0%, 70% 0%, 100% 50%, 70% 100%, 30% 100%, 0% 50%);

4. Masks (Transparency effects)

Linear Gradient Mask

mask: linear-gradient(45deg, transparent, black);

Radial Gradient Mask

mask: radial-gradient(circle, black 50%, transparent 70%);

5. Transforms (Position & rotation)

Rotate

transform: rotate(15deg);

3D Perspective

transform: perspective(500px) rotateY(25deg) rotateX(10deg);

6. Mix Blend Modes (Color mixing)

Multiply

mix-blend-mode: multiply;
background: #ff6b6b;

Screen

mix-blend-mode: screen;
background: #4ecdc4;

Difference

mix-blend-mode: difference;
background: #f9ca24;

7. Animations (Movement & effects)

Pulse Animation

animation: pulse 2s infinite;

@keyframes pulse {
0%, 100% { transform: scale(1); }
50% { transform: scale(1.05); }
}

Float Animation

animation: float 3s ease-in-out infinite;

Rotate Animation

animation: rotate 4s linear infinite;

8. Hover Effects (Interactive)

Zoom on Hover

transition: transform 0.3s ease;

.hover:hover {
transform: scale(1.1);
}

Blur on Hover

transition: filter 0.3s ease;

.hover:hover {
filter: blur(2px);
}

Tilt on Hover

transition: transform 0.3s ease;

.hover:hover {
transform: rotate(5deg) scale(1.05);
}
